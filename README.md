Free parking locator, community-contributed prototype for Unihack 2026 by team Hackathoners

This version is made assembled with HTML, JavaScript for formatting and front-end logic, and Python for SQL logic via SQLite and Flask. AI-driven in parts of the development, it has an external database hosted at supabase.com, with a website hosted at render.com. The map information is retrieved from the OpenStreetMap API.

Website to application: [text](https://fpa-cvn8.onrender.com/)
